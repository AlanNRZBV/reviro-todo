### Обзор проекта

Простой менеджер задач. Присутствует CRUD функционал, поиск и фильтрация. Поддержка тёмной, светлой и системной тем. Задачи, поиск(строка) и фильтрация сохраняются локально.

### Стек

React, vite, shadcnui, tailwind
